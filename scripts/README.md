# README

This folder contains scripts that are created by AI. This helps me a lot because I use tmux for my work flow. Instead of going into Obsidian (my documentation folder) and locating an amsi bypass, copy with my mouse and then paste it into the terminal with my mouse. It generates the amsi bypass and then I can use tmux bind key + "[" to go into copy mode, then press space to copy the contents. When completed I press "enter" and then bind key + "]" to paste the contents into a shell.
It is fast and effective.
